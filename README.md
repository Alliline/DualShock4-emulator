# DualShock4 emulator
Simple application to emulate the Sony DualShock 4 gamepad using an Xbox controller. This method is necessary for the fully work of the service [Sony Playstation Now](https://www.playstation.com/en-us/explore/playstation-now/). Works based on the driver [ViGEm](https://github.com/ViGEm).

## Setup
1. Install [ViGEmBus](https://github.com/ViGEm/ViGEmBus/releases).
2. Unpack and launch "DualShock4 emulator".
3. Launch PS Now or other app.

## Buttons
The "Back" button on the Xbox controller emulating pressing the touchpad on a Sony DualShock 4.

The "Share" button is assigned to the "F12" key.
